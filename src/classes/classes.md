# Классы

## Модус
Модус класса позволяет определить как персонаж взаимодействует с магией этого мира и как проявляет Волю для сопротивления божествам.

## Архетипы
Каждый персонаж должен выбрать себе один из четырёх архетипов. Архетипы отличаются использованием Воли и набором способностей. 

Архетипы не влияют на роль в группе - любой воин может стать магом или целителем, точно так же как любой маг или жрец может стать основным защитником группы.

Выбор класса влияет не на то чем будет заниматься персонаж, а на то, каким образом он будет это делать.

### Воин
Может использовать 1 Очко Воли чтобы добавить кость любой своей характеристики в пулл броска. Количество добавленных костей может быть неограничено.

Воины сосредоточены на прямолинейном и надёжном результате.

### Плут
Может потратить 1 Очко Воли чтобы добавить к броску пула ещё один результат. Количество добавленных результатов ограничено только пулом.

Плут самый азартный класс, часто ему требуется найти нестандартный подход для использования своих способностей на максимум.

### Маг
Может используя 1 Очко Воли увеличить ранг кости перед броском пула. Можно увеличивать ранг вплоть до d12, но только на тех костях, которые есть в пулле изначально.

Маги подходят тем игрокам, кто хочет почувствовать себя могущественными персонажами без гарантированного результата.

### Жрец
Может потратить 1 Очко Воли чтобы создать кость сцены d4. Максимальное количество костей не ограничено.

Жрецы сосредоточены на помощи всей партии, они обладают большим контролем над ситуацией чем остальные классы.

## Способности
Каждый класс имеет свой набор способностей. Персонаж может изучить любое количество способностей, но в один момент времени может использовать только 4. [Подробнее о способностях](../abilities/abilities.md).

## Специализации
Персонаж может изучить [специализацию](./specs.md) - особый пассивный навык, который влияет на все текущие и будущие изученные способности класса.
